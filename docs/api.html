Python Package

Wilson provides a python package that takes care of dealing with the serialization into the binary format and allows to handle the data on a higher level, e.g. by simply passing numpy arrays.

Example

 1import numpy as np
 2import wilson
 3
 4r = 5.0
 5t = np.linspace(0, 2*np.pi)
 6x = r * np.cos(t)
 7y = r * np.sin(t)
 8z = np.zeros_like(t)
 9
10path = np.column_stack((t, x, y, z))
11orbit = wilson.Path(path, name='Moon Orbit')
12
13earth = wilson.Sphere('Earth', description="That's our planet!", radius=1.0, color='blue')
14moon = wilson.Sphere('Moon', description='Our oldest pal!', radius=0.27, color='grey')
15moon.position = orbit
16
17project = wilson.Project('Earth Moon',
18   author='wilson',
19   description='The Moon orbiting around the Earth.',
20   animationSpeed=2*np.pi / 5.0,
21   animatables=[earth, moon])
22
23wilson.saveProject(project, '../examples/earth_moon.wlsn')

See notebooks for a detailed instructions.

ColorMap

A color map used to translate scalar values into colors by interpolating between discrete color stops. It is basically a table whose rows contains stops consisting of a scalar value followed by a color either as a rgb 3 tuple or rgba 4 tuple, i.e. an array of either shape (N,4) or (N,5).

Alternatively, one can specify the name of a colormap provided by matplotlib.

alias of Union[_SupportsArray[dtype], _NestedSequence[_SupportsArray[dtype]], bool, int, float, complex, str, bytes, _NestedSequence[Union[bool, int, float, complex, str, bytes]]]

ColorProperty

Color property can either be assigned a constant color either as a named color string (see xkcd colors), a normed rgb(a) tuple, a scalar value, or a array like of the shape (N,2) similar to scalar property. The constant scalar or the interpolated value from the GraphLike is converted into a color using the color map.

class Interpolation(value)

Data Interpolation modes

AHEAD = (2,)

Hold the next value until next data point is reached

HOLD = (1,)

Hold the last value until next data point is reached

LINEAR = (0,)

Linear interpolate between consecutive data points

STEP = (3,)

Hold last value until exactly in between two consecutive data points when it will hold the next value
