Python Package

Wilson provides a python package that takes care of dealing with the serialization into the binary format and allows to handle the data on a higher level, e.g. by simply passing numpy arrays.

Example

 1 import numpy as np
 2 import wilson
 3
 4 r = 5.0
 5 t = np.linspace(0, 2*np.pi)
 6 x = r * np.cos(t)
 7 y = r * np.sin(t)
 8 z = np.zeros_like(t)
 9
10 path = np.column_stack((t, x, y, z))
11 orbit = wilson.Path(path, name='Moon Orbit', interpolation='cubic')
12
13 earth = wilson.Sphere('Earth', description="That's our planet!", radius=1.0, color='blue')
14 moon = wilson.Sphere('Moon', description='Our oldest pal!', radius=0.27, color='grey')
15 moon.position = orbit
16
17 project = wilson.Project('Earth Moon',
18     author='wilson',
19     description='The Moon orbiting around the Earth.',
20     animationSpeed=2*np.pi / 5.0,
21     animatables=[earth, moon])
22 project.clearColor = 'light yellow'
23
24 wilson.saveProject(project, '../examples/earth_moon.wlsn')

Converts color specified by integers in the range 0-255 to the expected float tuple.
