Sidebar

The sidebar is on the left once a project is loaded and consists of several tabs:

Note

On top is a search bar allowing to filter inside different tabs.

Object Explorer

The object explorer shows all objects present in the scene including the overlay text ordered into groups. The groups can be separately enabled or disabled by clicking the eye symbol.

Path Explorer

Lists all paths present in the current project. By clicking on a path, they can be drawn in to the scene. It’s color can also be adjusted.

Settings Tab

The settings lists different options organized into groups. See Settings for more information.

Info Tab

The info tab shows the meta information of the current project.
